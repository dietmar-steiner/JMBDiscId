This library enables Java programms to calculate DiscIds for MusicBrainz.org and FreeDB.org and generate a submittion URL and a data lookup url for MusicBrainz.org
An example is provided in the examples package; Have a look there for reference.
to run the example just call: java JMBDiscId.jar <path to lib including lib name and extention> <drive path to use>
My cmd-line looks like this: java -jar "V:\Programming\Netbeans\JMBDiscId\dist\JMBDiscId.jar" "V:\Programming\Netbeans\JMBDiscId\libdiscid\discid.dll" "H:"
